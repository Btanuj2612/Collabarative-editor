# Collabarative-editor
# Collaborative Editor

A real-time collaborative text editor where multiple users can edit the same document together.  
All changes appear instantly for every connected user.

---

# Features

- Real-time text synchronization  
- Multiple users can collaborate together  
- Uses WebSockets for live updates  
- Lightweight and easy to use  
- No database required  

---

# Tech Stack

| Part | Technology |
|------|------------|
| Frontend | HTML, CSS, JavaScript |
| Realtime Communication | WebSockets / Socket.io |
| Backend | Node.js |


---

# How to Run the Project

```sh
# Clone the repository
git clone https://github.com/Btanuj2612/Collabarative-editor.git

# Go into the project folder
cd Collabarative-editor

# Install dependencies
npm install

# Start the development server
npm run dev
